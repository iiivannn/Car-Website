# NexDrive

A responsive landing page for a luxury electric vehicle dealership. Built as a mock-up project to practice frontend development with HTML, CSS, and JavaScript.

Designed with a mobile-first approach, featuring a clean layout, smooth animations, and enhanced user experience through ScrollReveal and Swiper.js.

**[View Live Demo](https://car-website-wheat.vercel.app/)**

## Features

- Responsive, mobile-first layout
- Hamburger menu navigation for mobile devices
- Scroll-based active link tracking
- Touch-enabled carousel for vehicle browsing
- Filterable product card grid
- Scroll-triggered animations

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Swiper.js (Carousel)
- MixItUp (Filtering)
- ScrollReveal (Animations)
- Remix Icon (Icons)
